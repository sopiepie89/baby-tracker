How to get Firebase URL · 如何获取
  1. Go to firebase.google.com · 前往 Firebase 网站
  2. Sign in → Add project → name it baby-tracker
  3. Click Build → Realtime Database
  4. Create Database → Test mode → Enable
  5. Copy the URL shown at the top (ends in .firebaseio.com)
  6. Paste it above and tap Save · 粘贴到上方并保存
